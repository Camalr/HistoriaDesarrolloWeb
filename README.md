# HistoriaDesarrolloWeb
NUDO
La rata se alzó diciendo que sería mejor que todos ganaran lo mismo
Afirmó que la igualdad traería justicia, y pidió manejar la economía
Con un ejército de ratas fue tomando el control poco a poco
A sus seguidores les prometió beneficios, si se mantenían fieles
Mientras tanto, comenzó a reprimir a los demás animales
El esfuerzo dejó de ser recompensado, y muchos dejaron de producir
La escasez apareció, y el miedo se volvió común en la granja
Con los años, el estado de la granja fue decayendo cada vez más.
