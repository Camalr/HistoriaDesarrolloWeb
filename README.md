# HistoriaDesarrolloWeb

INICIO
Había una vez una granja que era próspera y bien organizada, donde todos convivían
Cada animal aportaba según lo que sabía hacer, y ganaba una porción justa para vivir
El trabajo tenía sentido, porque el esfuerzo se veía reflejado en resultados
La producción crecía, y la granja avanzaba sin mayores conflictos
Había respeto entre los animales, y confianza en cómo funcionaban las cosas
Nadie necesitaba mandar con miedo, porque el orden era natural
Cada día la granja demostraba que podía seguir mejorando
Hasta que un día, una rata decidió que ese sistema no le convenía.
